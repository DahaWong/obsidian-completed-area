## 使用
- 在主页左侧菜单栏点击 Footlinks 的图标.
- 或者**打开指令面板**（Cmd/Ctrl + p） 输入`Footlinks`

## 设置
在 `Footlinks` 设置页面，可以:
- 在正文和尾链之间添加一个分割线`---`（默认为空）
- 决定是否在主页左侧菜单栏中显示图标。
